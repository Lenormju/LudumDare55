# My Occult Therapist

You are the worst therapist in the world. Most of your patients have found you infuriating, petty or disagreeable. But this may be your best assets for an other kind of clients ... :imp:

# Trigger warnings :warning:

This game features demonic creatures, so please be advised :
- if you are a human, it contains crude language about sex, violance, self-harm and other things
- if you are a demon, it contains vivid descriptions of happiness, kindness, work satisfaction, ...

# How to play :keyboard: :video_game: 

- either with the keyword : directional arrows + Enter or Space
- or the mouse : click left + click right
- in full screen :computer:
